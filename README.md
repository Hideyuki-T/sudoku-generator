# ディレクトリ構成:
```
sudoku-generator/
├── python/
│   └── main.py           # Python実装
├── java/
│   └── Main.java         # Java実装
├── cpp/
│   └── main.cpp          # C++実装
├── javascript/
│   └── main.js           # JavaScript実装
├── php/
│   └── index.php         # PHP実装
├── ruby/
│   └── main.rb           # Ruby実装
└── README.md             
```

# 使い方
- pythonコマンド
python3 main.py

- javaコマンド
java Main

- cppコマンド
g++ -std=c++11 -o sudoku main.cpp
コンパイル成功後↓
./sudoku
