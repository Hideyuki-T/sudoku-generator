# ディレクトリ構成:
```
sudoku-generator/
├── python/
│   └── main.py           # Python実装
├── java/
│   └── Main.java         # Java実装
├── cpp/
│   └── main.cpp          # C++実装
├── javascript/
│   └── main.js           # JavaScript実装
├── php/
│   └── index.php         # PHP実装
├── ruby/
│   └── main.rb           # Ruby実装
└── README.md             
```

# 使い方
| 言語         | 実行コマンド                            |
|------------|-----------------------------------|
| Python     | python main.py                    |
| Java       | java Main                         |
| C++        | g++ -std=c++11 -o sudoku main.cpp |
|            | ./sudoku                          |
| JavaScript | node main.js                      |
| PHP        |                                   |
| Ruby       |                                   |
