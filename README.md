# 概要：
6つの言語で挑戦！Python から Ruby まで、数独ジェネレーターを実装する！

# ディレクトリ構成:
```
sudoku-generator/
├── python/
│   └── main.py           # Python実装
├── java/
│   └── Main.java         # Java実装
├── cpp/
│   └── main.cpp          # C++実装
├── javascript/
│   └── main.js           # JavaScript実装
├── php/
│   └── index.php         # PHP実装
├── ruby/
│   └── main.rb           # Ruby実装
└── README.md             # 優雅なる総合案内書
```

# 使い方
- pythonコマンド

python3 main.py

- javaコマンド

java Main
